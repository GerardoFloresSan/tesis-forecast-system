Métricas del modelo
  • MAE i Mean Absolute Error. Mide el error promedio absoluto del modelo. En este caso, el pronóstico se equivoca en promedio en alrededor de {{ summary.lstm_metrics.mae | number:'1.2-2' }} interacciones. Error promedio por predicción
    {{ summary.lstm_metrics.mae | number:'1.2-2' }}
  • RMSE i Root Mean Squared Error. Da más peso a errores grandes. Aquí indica que cuando el modelo falla más de lo normal, el error puede acercarse a {{ summary.lstm_metrics.rmse | number:'1.2-2' }} interacciones. Impacto de errores grandes
    {{ summary.lstm_metrics.rmse | number:'1.2-2' }}
  • MAPE i Mean Absolute Percentage Error. Representa el error porcentual promedio del modelo. Actualmente muestra una desviación aproximada de {{ summary.lstm_metrics.mape | number:'1.2-2' }}% respecto al valor real. Error porcentual promedio
    {{ summary.lstm_metrics.mape | number:'1.2-2' }}%
  • R² i Coeficiente de determinación. Indica cuánto del comportamiento de la demanda logra explicar el modelo. En este caso, explica aproximadamente el {{ (summary.lstm_metrics.r2 * 100) | number:'1.0-0' }}% de la variabilidad. Capacidad explicativa del modelo
    {{ summary.lstm_metrics.r2 | number:'1.2-2' }}
  • Train size i Cantidad de registros utilizados para entrenar el modelo y enseñarle patrones históricos. Datos usados para entrenamiento
    {{ summary.lstm_metrics.train_size }}
  • Test size i Cantidad de registros utilizados para validar el modelo con datos que no vio durante el entrenamiento. Datos usados para validación
    {{ summary.lstm_metrics.test_size }}
